# 宠物视频 → Electron 桌宠 替换流程

## 输入

我会给你一个宠物视频的**完整路径**。

## 任务

基于已有项目模板，提取视频中的宠物头部动作，**仅替换宠物视觉资源**，所有功能、设置、交互方式保持不变。

## 替换步骤

### 第 1 步：提取视频帧

用 ffmpeg 提取所有帧，保持原始帧率。视频建议 5 秒左右、30fps。

`bash
ffmpeg -i "视频路径.mp4" -start_number 0 frames/frame_%04d.png
`

### 第 2 步：AI 抠图（保留毛发细节）

使用 rembg 逐帧去除背景，输出 RGBA 透明 PNG。

`python
from rembg import remove
from PIL import Image
for frame in frames:
    output = remove(frame)
    # 保存为带透明通道的 PNG
`

### 第 3 步：缩放到统一尺寸

每帧缩放到 **160×160 像素**（使用 LANCZOS 采样保证质量）。

### 第 4 步：拼成精灵图

拼成 **10 列 × 15 行 = 总共 150 帧** 的精灵图，输出到 
esources/spritesheet.png。

`python
sheet = Image.new('RGBA', (10 * 160, 15 * 160), (0,0,0,0))
for i, frame in enumerate(frames):
    col = i % 10
    row = i // 10
    sheet.paste(frame, (col * 160, row * 160), frame)
sheet.save('resources/spritesheet.png')
`

### 第 5 步（可选）：替换图标

用第一帧裁剪缩放到 256×256，替换 
esources/icon256.png，然后用 rcedit 写入 exe：

`bash
npx rcedit "小皮休.exe" --set-icon "resources/icon256.png"
`

## 精灵图规范（严格遵循，不可修改）

| 属性 | 数值 |
|------|------|
| 总帧数 | 150 |
| 布局 | 10 列 × 15 行 |
| 每帧宽高 | 160 × 160 像素 |
| 图像模式 | RGBA（透明通道） |
| 文件路径 | 
esources/spritesheet.png |
| 帧序号 → 角度 | 帧0 = 0°（正面），帧75 = 180°（背面），均匀映射到 360° |

## 功能清单（替换后必须全部保留）

- 透明窗口、无边框、始终置顶
- 默认在桌面右下角
- 全局鼠标追踪，猫头 360° 平滑跟随
- 圆形帧插值（无跳变、无卡顿）
- 鼠标靠近时平滑回到 idle 正面
- 右键猫 → 菜单：锁定/解除 / 开机自启动 / 退出
- Lock 模式 CSS pointer-events 穿透
- Unlock 模式可拖拽到任意位置
- 系统托盘菜单（与右键菜单一致）
- 支持 start.bat 双击启动
